A 6-lab hands-on workshop demonstrating the full journey of building an AI agent with Amazon Bedrock AgentCore — from a local prototype to a production-grade system. Using a Customer Support Agent example built with Strands Agents and Amazon Nova 2 Lite, this workshop covers adding memory and personalization, sharing tools securely via Gateway & Identity, deploying to production with observability, evaluating agent performance, and building a customer-facing UI. Concepts apply to any agent framework or model.
